## NiftyPM

| Action                               |    Shortcut     |
| :----------------------------------: | :-------------: |
| Create a Task                        | `⌘` + `⌥` + `t` |
| Create a Milestone                   | `⌘` + `⌥` + `m` |
| Creating a Document                  | `⌘` + `⌥` + `o` |
| Editing a Message or Comment         |       `↑`       |
