# Todoist Keyboard Shortcuts

### Task Quick Add Shortcuts

| **Shortcut**           | **Action**           |
|:----------------------:|:--------------------:|
| `@`                    | Add a label          |
| `#`                    | Pick a project       |
| `+`                    | Add an assignee      |
| `p1`, `p2`, `p3`, `p4` | Set a priority level |
| `⌃` `m`                | Add a quick comment  |

### Used Anywhere

| **Shortcut**                 | **Action**                 |
|:----------------------------:|:--------------------------:|
| `⌃` `Click` _or_ `⇧` `Click` | Bulk edit tasks            |
| `q`                          | Add task                   |
| `a`                          | Add new task at the bottom |
| `A`                          | Add new task at the top    |
| `/` _or_ `f`                 | Open search                |

### Adding or editing tasks

| **Shortcut** | **Action**                                       |
|:------------:|:------------------------------------------------:|
| `Esc`        | Cancel current changes                           |
| `↵`          | Save new task and create a new one below it      |
| `⇧` `↵`      | Save existing task and create a new one below it |
| `⌃` `↑`      | Move 1 task up                                   |
| `⌃` `↓`      | Move 1 task below                                |

### Creating and completing sub-tasks

| **Shortcut** | **Action**           |
| ------------ | -------------------- |
| `⌘` `→`      | Increase task indent |
| `⌘` `←`      | Decrease task indent |

### Sorting tasks inside a project

| **Shortcut** | **Action**       |
| ------------ | ---------------- |
| `d`          | Sort by date     |
| `p`          | Sort by priority |
| `n`          | Sort by name     |
| `r`          | Sort by assignee |
