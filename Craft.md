## Craft Keyboard Shortcuts

### Basic Navigation

**Action** | **Shortcut**
:-----:|:-----:
Navigate Back | `⌘` + `←` or `⌘` +  `[` or `⌘` + `⌥` + `←` 
Navigate Forward | `⌘` + `→` 
Navigate Into | `⌘` + `⌥` + `→` or `⌘` + `⌥` + `]` 
Quick Open | `⌘` + `O` 
Go to Daily Note | `⌘` + `⌥` + `N` 
Open Link | `⌘` + `⌥` + `Return` 

### Documents and Search

**Action** | **Shortcut**
:-----:|:-----:
New Document | `⌘` + `N` 
New Quick Note | `⌘` + `⌥` + `N` 
Delete Document | `Delete`
Search in Space | `⌘` + `F` 
Copy Deeplink to Current View or Page | `⌘` + `⌥` + `L` 
Copy Deeplink to Selected Document or Block | `⌘` + `⌥` + `⇧` + `L` 

### Tasks

**Action** | **Shortcut**
:-----:|:-----:

Mark Task as Canceled | `⌘` + `⌥` + `⇧` + `T` 
Mark Task as Done | `⌘` + `⌥` + `T` 
Mark Task as Canceled | `⌘` + `⌥` + `⇧` + `T` 
Create New Task | `Space`
Navigate Tasks | `↑` / `↓`
Open Task Details | `Return`
Return to Task List | `Esc`
Navigate Dates | Arrow keys
Select Date | `Return`

### Window and Tabs

**Action** | **Shortcut**
:-----:|:-----:
New Window | `⌘` + `⇧` + `N`

New Tab | `⌘` + `T` 
Switch to Tab 1-9 | `⌘` + `1` through `⌘` + `9`

Reopen Last Closed Tab | `⌘` + `⇧` + `T`
